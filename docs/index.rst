*********************
Google Cloud IoT Core
*********************

Cloud IoT Core is a fully managed service that allows you to easily and securely connect, manage, and ingest data from millions of globally dispersed devices: `Google Cloud IoT Core <https://cloud.google.com/iot-core/>`_

It is possible, for example, to connect to Google Cloud IoT Core `MQTT <http://mqtt.org/>`_ Broker to send device data and process/store those data with different Google Cloud services linked to Google Cloud IoT Core through `Google Cloud Pub/Sub <https://cloud.google.com/pubsub/docs/overview>`_.
